## Installation

Add this line to your Jekyll site's `Gemfile`:

```ruby
gem "jekyll-theme-jam"
```

And add this line to your Jekyll site's `_config.yml`:

```yaml
theme: jekyll-theme-jam
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install jekyll-theme-jam
```
